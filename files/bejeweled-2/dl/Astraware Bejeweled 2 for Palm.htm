How To Install

Use an extractor program such as WinZip or Stuffit Expander to get at the Bejeweled 2 files.

Install the game by double-clicking the .prc file, which opens the Palm Install Tool or Quick Install program.

If one is available, users with Palm OS® 5 devices may wish to install the music file as well. Double click on the .pdb file (or files, if there is more than one) to install it in the same way as installing the .prc file. Please note that not all Palm OS® 5 handhelds support music, for more information please refer to our knowledgebase here.

Use the HotSync button as usual to install the game onto your handheld. If you have a handheld with an expansion card, you can then move the game over to it. Do this before you register the game!

Now run Bejeweled 2 and tap the screen to play the game!

How to Play

Bejeweled 2 uses the same classic gameplay from Bejeweled with a few unique twists, and of course the addition of some explosive new game modes!

For those new to the Bejeweled phenomenon, the premise is simple - the screen will fill with a random arrangement of seven different gem types. The aim is to try and make lines (vertical or horizontal) of 3 or more identical gems by swapping pairs of gems. As you make matches and remove gems, new gems will drop in from above.

Swapping a pair of gems can only occur if the swap will result in a match occurring, if it does not, the gems will return to their original positions. You cannot swap gems diagonally, only horizontally and vertically!

In Bejeweled 2, making a line of 4 gems will produce a Power Gem. These powerful gems are left behind, and if used to create a match as normal, will explode, destroying all the gems around them! These can be great for making some space in the game board and getting new gems into place.

Making a line of 5 gems is the ultimate in experiences - this creates the rare and powerful Hyper Cube. Hyper Cubes can always be swapped with a gem, even if the swap would not make a match...the reason for this is that when swapped, the Hyper Cube activates, and lightning shoots forth to destroy all the gems of that type on the board!

Game Modes

Bejeweled 2 features a host of games modes, from the classic mode we all know and love, to the mind bending Puzzle mode! There are also several additional game modes that you can unlock as you play the game - can you find them all?

Classic

Classic mode is based on the game which started it all, Bejeweled itself. You have unlimited time to make matches, but if you run out of moves it's game over! Make careful use of Power Gems and Hyper Cubes to keep the gems flowing!

You can always ask for a hint by tapping on the "?" button in the bottom left, but this will cost you a little of your progress towards the next level.

Action

In Action mode, speed is of the essence! You will always have at least one move available to you, but the timer bar will get progressively faster as time passes. Make matches quickly to fill the timer bar, and when it reaches the end you will progress to the next level!

Requesting a hint in Action mode will cost you a little time from the Timer bar, so keep an eye on it.

Endless

As the name implies, Endless mode keeps going forever! Endless mode will always ensure that you have at least one move available, and there is no way to lose in this mode. As you progress through the levels, you will collect gems - maybe something special will happen when you have enough?

Puzzle

Puzzle mode is very different from the normal Bejeweled modes. Instead, there will be an arrangement of gems on the screen, and your goal is to make matches to remove all the gems! This may take several moves and careful planning to achieve!

Special Gem Types

The different gem types you will encounter in Bejeweled 2 are as follows:

  • Normal Gems : The normal classic gems, and the mainstay of Bejeweled 2. Make matches to remove them and score points!
  • Power Gems : Power Gems are characterised by a white glow emanating from their centre. Create these by making a match with 4 identical gems. They can then be used in another match to create a powerful explosion!
  • Hyper Cube : The ultimate in gems, the philosopher's stone. Hyper Cubes can be created by making a match with 5 identical gems. Hyper Cubes can be matched with any other gem, and when this is done, will remove all the gems of that type from the board. Hyper Cubes can be seen as a multicoloured swirl of energy.
  • Rocks : Rocks are unreactive lumps of...rock. You cannot form matches with Rocks, although they can be swapped around as normal. To destroy Rocks, use a Power Gem, Time Bomb or Hyper Cube!
  • Time Bombs : Time Bombs are black and spiky, with a counter on their front. As you make matches, this counter will go down. When it reaches 1, the bomb will start shaking to show that it's about to go off. The next match you make will cause the bomb to explode, destroying all the gems around it!

Game Options

The Tips Game option allows you to determine whether or not to show the dialog boxes containing hints and tips about how to play Bejeweled 2.

The Hint Sparkles controls whether the automatic "hint" system will be active. If this is enabled (the default), then if you make no move for a few seconds, then a valid move will be indicated by the gem you need to move sparkling. Unlike asking for a hint, the automatic hint system will never cost you points/time.

The Transparency slider under the Graphics settings allows you to alter the transparency of the game board. Setting it to the far left will give a completely transparent board, while the right edge of the slider will produce an opaque board.

Screensaver Settings

The Select Screensaver button allows you to change which mode the Screensaver will use from: None, Demo (automated play), and Slideshow (will cycle through the different backgrounds). Please note that the Slideshow setting does not appear in Bejeweled 2 Lite, as the Lite version only has the one background to reduce the filesize and memory requirements.

The Slideshow speed setting allows you to control how quickly the background will switch to a new one in Slideshow mode.

The Select Clock button allows you to set whether the Screensaver clock appears or not (Default: On). If the clock is enabled, it will appear in the top right corner, and allow you to move it around the screen by dragging it with the stylus! You can also use the ">" button to advance the Screensaver to the next screen (either a new set of gems, or a new background depending on the mode). The "X" button allows you to quit from the Screensaver mode and return to the main menu.

Start Screensaver allows you to manually start the Screensaver. You can only use this option while on the title screen, the Screensaver may not be enabled while in-game.

Hidden Features

Bejeweled 2 features several hidden game modes...

Original

Original mode is as the name implies, original Bejeweled with absolutely none of the new special gems. To unlock it, play Classic mode extensively!

Hyper

The speed of Hyper mode defies human description, but otherwise plays in the same way as Action mode. To unlock it you will need lightning reflexes built up from playing Action mode to high levels.

Cognito

Cognito mode is only available to those who can answer every puzzle posed to them! In Cognito mode, the puzzles are mirrored from their Puzzle mode equivalents, the colours are different and you score points for each move you make! Take heed, however: when you ask for a hint in Cognito mode, you will not score any further points for the rest of that puzzle.

Finity

Finity mode can be unlocked by the truly tenacious in Endless mode. In Finity mode, you have a timer bar much like in Action mode, but power gems, rocks and timebombs can drop in from the top as well as normal gems! Further, in Finity mode, single matches score no points - only chain reactions (also known as "cascades") will score points!

Twilight

To enter Twilight mode, circle the stylus clockwise around the purple mode buttons on the title screen, touching each in turn. You may need to circle several times before it activates. Twilight mode is very similar to Classic mode, except that gems can fall from both the top and the bottom of the screen!

Registering

In Demo form, you can play 5 games of Bejeweled 2 for free, as you decide whether you like it or not. After these games are up, the program will display a short message reminding you to register each time you play!

If you beam a demo to someone else, and they also like the game, then they can register it either by purchasing a copy from a local store or by purchasing online.

Registering Online:

For those that like the speed and convenience of buying via the internet, when you register for the game you'll get a code which will unlock the game and remain valid for future upgrades. Go to the web page at http://www.astraware.com/palm/bejeweled2/ for more details.

You'll need to know your HotSync ID which you can find by running Bejeweled 2, opening the menus, and selecting the 'Register...' option. Your HotSync ID will be filled in for you below.

When you register, Astraware will generate a key based on your ID, which will normally be sent out by email within 48 hours of registering. When you receive your key, go to the same register option, and enter the key into the space provided.

Release Revision History!

Release 1.2 : 1st April, 2005
  • Fixed an issue relating to switching levels in the Galaxy map
  • Fixed a bug which could cause some of the time when a device is switched off to be counted towards time spent in game
  • Fixed an issue which could cause the game state to be lost if an "Insufficient heap memory" error message was displayed at startup
  • Fixed an bug where renaming a profile would default to a blank name, rather than the existing name
  • Tweaked the probability of large "floaty" scores appearing to make them more common
  • Added an option to return to the title screen from the "You must register to continue" dialog box
  • Added a transparency slider to control the transparency of the game board
  • Added screensaver mode
    • Demo play and Slideshow modes (Please note: Slideshow mode is not available in the "Lite" version of Bejeweled 2, as it requires the backgrounds which were removed to reduce the file size and memory requirements)
    • Screensaver can be launched deliberately from the Screensaver menu option, or by leaving the device on the game selection screen for several seconds
    • Optional digital clock display in Screensaver mode, which may be moved anywhere on the screen!
Release 1.1 : 6th January, 2005
  • Fixed an issue regarding savegames and stats being reset if the game was unable top start due to low memory
  • Fixed an issue with hypercube generation with "T" shape gem arrangements
  • Improved stylus dragging performance on the Tungsten T5 and Treo 650
  • Added Bejeweled 2 Lite!
    Bejeweled 2 Lite differs from Bejeweled 2 in the following ways:
    • Reduced number of graphics
    • Reduced in-game effects
    • Greatly reduced memory requirements
  • Users who have experienced issues getting Bejeweled 2 to run on their device due to low memory, or who wish to save some space, should install Bejeweled 2 Lite instead of the standard Bejeweled 2!
Release 1.0 : 6th December, 2004
  • First Public Release!
